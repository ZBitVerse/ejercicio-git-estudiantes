# ejercicio-git-estudiantes
Repositorio GIT para ejercicio en clase

1. Por favor agregue un archivo de texto con nombre así:  SuNombre.txt
2. Dentro del archivo coloque un párrafo de presentación, indicando al menos lo siguiente:
    - dónde esta radicado
    - Profesión/titulos
    - A qué se dedica
    - Pataformas de desarrollo de SW en las que tiene experiencia
3. Suba el archivo al repositorio
